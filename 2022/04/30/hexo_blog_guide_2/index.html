# 配置 github 仓库

前往 github 注册一个账户

然后进入主页，点击右上角 + 号，选择 New repository

找到 Repository name 那一栏，填上

点击页面最下方 Create repository 创建仓库

打开本地博客根目录，修改_config.yml 文件，在文件末尾插入以下片段

# 注册域名和服务器

注册自己喜欢的域名和服务器，然后进入控制面板，选择域名解析，将 CNAME 地址绑定 username.github.io 链接

# 域名绑定仓库

进入刚才创建的仓库页面，点击右上角 Settings 按钮

选择 General 一栏下的 Branches，点击右边的铅笔按钮，将当前分支更名为 master

选择 General 一栏下的 Pages 一栏，点击 Choose a theme 按钮选择主题，然后在 Custom domain 一栏输入刚才申请的域名，点击 Save 按钮
